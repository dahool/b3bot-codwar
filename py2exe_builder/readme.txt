This folder holds the tools and scripts to build a .exe release of B3
and an installer.

REQUIREMENTS:
    Environment :
        * a Windows OS
        * with python2.6+
        * with mysqldb
    Tools:
        * py2exe2.6+
        * InnoSetup
        * InnoSetup preprocessor
        * ISTools (optional)
    
HOW TO BUILD THE .EXE RELEASE and distributions :
    * check the build.yaml config file
    * run python.exe build.py
